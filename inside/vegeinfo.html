Vigenère Cipher

The Vigenère cipher is a method of encrypting alphabetic text by using a simple form of polyalphabetic substitution. A polyalphabetic cipher uses multiple substitution alphabets to encrypt the text, making it more secure than simple substitution ciphers.

Here's how the Vigenère cipher works:

  • Select a keyword (e.g., "KEY").
  • Repeat the keyword to match the length of the plaintext message (e.g., "KEYKEYKEY...").
  • For each letter in the plaintext, shift it forward by the corresponding letter in the keyword (e.g., 'K' shifts by 10, 'E' by 4, 'Y' by 24).
  • Wrap around the alphabet if necessary.
  • Encrypt the message by replacing each letter with the shifted one.
    • Example:

    Plaintext: "HELLO"

    Keyword: "KEY"

    Repeated Keyword: "KEYKEYKEY"

    Encrypted Message:

    "H" is shifted by "K" to become "K".
    "E" is shifted by "E" to become "E".
    "L" is shifted by "Y" to become "I".
    "L" is shifted by "K" to become "W".
    "O" is shifted by "E" to become "X".
    So, the encrypted message is "KEIWX".
    Decryption is performed by shifting the letters in the reverse direction using the same keyword.
